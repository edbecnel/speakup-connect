# Administrator Tutorial

Use this tutorial in sequence for first-time administrators and delegated staff.

---

## Module 1 - Admin orientation

1. Open **Settings**
2. Confirm **Administration** section visibility
3. Identify which admin tools your account can access

---

## Module 2 - Join and member workflow

1. Open **Join Applications**
2. Review one request
3. Approve or reject
4. Open **Member Management** and verify member status

---

## Module 3 - Group operations

1. Open **Groups & Clubs (Administration)**
2. Open or create a group
3. Add one member or review one request

---

## Module 4 - Publishing workflow

1. Draft one announcement
2. Draft one alert
3. Verify behavior when approval is enabled

---

## Module 5 - Roster and grade tools

1. Open **Student Roster**
2. Review one student profile
3. Open **School Grades** and verify grade setup

---

## Module 6 - Translation basics

1. Open **Translations**
2. Review one key in target language
3. Save or approve one entry (if permitted)

---

## Module 7 - Help Center completion

1. Open **Settings -> Help Center**
2. Open **Administrator Guide** for full reference content
3. Keep this tutorial for onboarding new admin staff
