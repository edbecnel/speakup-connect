# Member Tutorial

Use this tutorial in order. It is designed for first-time users and can be completed fully in the app.

---

## Module 1 - Sign in

1. Open **Sign In**
2. Enter student ID or contact email
3. Enter password
4. Confirm you can open Home

---

## Module 2 - Home and language

1. Open **Home**
2. Use the **globe** language picker
3. Confirm the same option in **Settings -> Appearance -> Language**

---

## Module 3 - Submit a report

1. Open **Submit a Concern**
2. Fill required fields
3. Submit
4. Check **My Reports** for status tracking

---

## Module 4 - Alerts and responses

1. Open **Alerts**
2. Read one alert
3. If prompted, submit a response

If response is required, complete it before dismissing.

---

## Module 5 - Groups and clubs

1. Open **Browse Groups & Clubs**
2. Request to join one group (if enabled)
3. Open **My Groups & Clubs** and review your membership list

---

## Module 6 - Help and support

1. Open **Settings -> Help Center**
2. Open **Member Guide** for reference details
3. Note who to contact for school policy or account issues
